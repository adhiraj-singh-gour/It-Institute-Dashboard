## 🚀 Project Overview

This project is a **Power BI dashboard** designed for an IT Training Institute to track **student performance and placement readiness** across multiple batches and courses.

The main goal of this project is to:

* Analyze student performance (Attendance, Tests, Mock Interviews)
* Identify **top students for placement**
* Help management make **data-driven decisions**

---

## 🎯 Problem Statement

In training institutes, there are multiple batches and many students, but:

* Not all students are ready for placement
* Manual evaluation is time-consuming
* No centralized performance tracking system

👉 This dashboard solves these problems using data.

---

## 💡 Solution

We created a **Placement Decision System** using Power BI that:

* Combines data from multiple batches
* Calculates a **Final Performance Score**
* Categorizes students based on readiness
* Provides interactive filtering and insights

---

## 📂 Dataset Information

The dataset includes:

* Student Details (Name, Batch, Course, Trainer)
* Attendance Data
* Test Scores
* Mock Interview Scores
* Fees Status

---

## 🧮 Performance Calculation

Final Score is calculated using:

Final Score = (0.3 × Attendance) + (0.4 × Test Score) + (0.3 × Mock Interview)
